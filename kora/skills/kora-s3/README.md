# Kora S3

S3-compatible object storage: AWS S3, MinIO, file upload/download, presigned URLs.

## When to Use

- AWS S3 integration
- MinIO local storage
- File upload/download
- Presigned URLs (temporary access)

## Quick Start

```bash
/kora-s3 --provider aws --bucket my-bucket
```

## Key Features

- AWS S3 integration
- MinIO compatibility
- File upload/download
- Presigned URLs
- Configuration: endpoint, credentials, region

## Triggers

S3, AWS, MinIO, file upload, presigned URL, bucket, object storage
