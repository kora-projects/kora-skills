# Kora MapStruct

Object mapping with MapStruct: DTO ↔ Entity, custom mappers, Kora integration.

## When to Use

- DTO ↔ Entity mapping
- Custom mappers (@Mapper)
- Integration with Kora DI
- Collections, null-safe mapping

## Quick Start

```bash
/kora-mapstruct --mapper UserMapper --source User --target UserDto
```

## Key Features

- @Mapper, @Mapping, @Mappings
- DTO ↔ Entity mapping
- Custom methods (default, named)
- Integration with Kora @Component
- Collections, Optional, null-safe

## Triggers

MapStruct, @Mapper, DTO mapping, Entity mapping, custom mapper, @Component
