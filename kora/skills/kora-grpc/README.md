# Kora gRPC

gRPC clients and servers in Kora. protobuf-gradle-plugin, @Component handlers, interceptors.

## When to use

- gRPC servers with @Component handlers
- gRPC clients (ClientInterceptor)
- gRPC Server Reflection
- Protobuf contracts

## Quick start

```bash
/kora-grpc --service UserService --package com.example
```

## Key features

- grpc-client / grpc-server modules
- protobuf-gradle-plugin 0.9.4
- @Component handlers
- ClientInterceptor / ServerInterceptor
- gRPC Server Reflection

## Triggers

gRPC, protobuf, grpc-server, grpc-client, ClientInterceptor, ServerInterceptor, reflection
